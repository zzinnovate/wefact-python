# Installation

## Prerequisites

- Python 3.11+
- pip
- requests

## From PyPI

```bash
pip install wefact-python
```

## From source

```bash
git clone https://github.com/zzinnovate/wefact-python.git
cd wefact-python
pip install .
```

## Verify

```python
import wefact
print(wefact.__version__)
```

If you see a version string, the package is installed correctly.
